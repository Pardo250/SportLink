Actor principal: Usuario autenticado
-Descripción: El usuario debe seleccionar un método de pago para confirmar su reserva.
-Precondiciones: Haber completado zona, deporte y horario.
-Postcondiciones: El usuario elige un método de pago válido.

Flujo principal:

1. El sistema carga los métodos de pago disponibles.

2. El usuario selecciona uno.

3. El sistema guarda esta selección en la reserva.
