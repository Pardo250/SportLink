Actor principal: Usuario autenticado
-Descripción: El usuario consulta las zonas donde se pueden reservar espacios deportivos.
-Precondiciones: El usuario debe haber iniciado sesión.
-Postcondiciones: El sistema muestra todas las zonas registradas.

Flujo principal:

1. El usuario ingresa a la vista “Reservar”.

2. El sistema carga todas las zonas desde la base de datos.

3. El usuario selecciona una zona para continuar con la reserva.
