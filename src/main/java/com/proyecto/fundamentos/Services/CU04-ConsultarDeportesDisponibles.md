Actor principal: Usuario autenticado
-Descripción: El usuario elige el deporte para el cual desea reservar un espacio.
-Precondiciones: Haber accedido a la sección de reservas.
-Postcondiciones: El usuario selecciona un deporte.

Flujo principal:

1. El sistema carga la lista de deportes disponibles.

2. El usuario selecciona uno de manera obligatoria.
